Microsoft Azure

Microsoft Azure is an internet-scale cloud services platform hosted in Microsoft datacenters, which provides an operating system and a set of developer services that can be used individually or together. Azure’s interoperable platform can be used to build new applications to run from the cloud or enhance existing applications with cloud-based capabilities.
